# EcoBosque Hotel - Design System

## Overview
Eco-tourism hotel management system. Warm, organic aesthetic inspired by forest and nature.

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `primary` | `#2D5A3D` | Forest green - buttons, active states |
| `primary-light` | `#4A7A5E` | Hover states |
| `primary-bg` | `#E8F0E8` | Light backgrounds |
| `secondary` | `#5B7FA3` | Blue - reservada state |
| `accent` | `#B8860B` | Gold - ocupada state |
| `surface` | `#F8FAF8` | Card backgrounds |
| `surface-warm` | `#F5F4F2` | Warm neutral |
| `text` | `#1E2533` | Primary text |
| `text-muted` | `#6B7280` | Secondary text |
| `border` | `#E5E7EB` | Borders |

## Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headings | `Gilda Display` | 400 | 24-32px |
| Body | `Barlow` | 400-500 | 14-16px |
| Labels | `Barlow Condensed` | 600-700 | 12-14px |

## Effects

- **Transitions**: 200-300ms ease
- **Border-radius**: 8-12px (cards), 6-8px (buttons)
- **Shadows**: Soft, subtle (`0 2px 8px rgba(0,0,0,0.08)`)

## Anti-patterns

- ❌ AI purple/pink gradients
- ❌ Harsh animations
- ❌ Neon colors
- ❌ Dark mode (for now)
